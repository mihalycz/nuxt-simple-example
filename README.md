### Run developer environment

1. Go to front app directory
 
    ```bash
    cd front

2. Copy and edit .env
 
    ```bash
    cp .env.dist .env && nano .env

3. Install dependencies
 
    ```bash
    npm i

4. Run the site dev version
 
    ```bash
    npm run dev
